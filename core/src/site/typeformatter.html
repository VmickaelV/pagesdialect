TypeFormatter interface

Custom formatting and sorting

You can customize the formatting in the exported PDF and Excel files implementing the interface TypeFormatter.
The formatter has access to the HttpServletRequest object, so it can use request data like Locale.

You can find an example of implementation in the webapp demo, search for class DateFormatter.
